# Ayush Singh — Portfolio Website

Personal portfolio website built with **Vite + React + Tailwind CSS**.

🔗 **Live:** [erayushs.github.io/my-website](https://erayushs.github.io/my-website) *(deploy to update)*

---

## Tech Stack

- **Vite 8** — lightning-fast dev server & build tool
- **React** (JavaScript) — component-based UI
- **Tailwind CSS v4** — utility-first styling via `@tailwindcss/vite`
- **Lucide React** — icon library
- Custom scroll-reveal with `IntersectionObserver`

## Features

- Dark editorial design with amber/gold accents
- Cormorant serif display font + Outfit body font
- Typewriter role animation in hero
- Scroll-triggered fade-up animations on all sections
- Fully responsive (mobile-first)
- Sections: Hero · Stats · Experience · Projects · Skills · Contact

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Build for production:

```bash
npm run build
```
